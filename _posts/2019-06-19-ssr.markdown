# SSR(Server Side Rendering)
======
-서버에서 처리해야 할 일들을 최대한 줄이고, JS로 처리할 수 있는 부분으로 처리
-초기 페이지 로드가 클라이언트 사이드 렌더링보다 빠르다.

# SPA(Single Page Application)
=====
-초기 구동 속도가 느리다.
-그러나 한번 구동하고 나면 훨씬 더 좋은 사용성을 느낄 수 있다.
-오래된 브라우저(IE 8 이하)에서는 오류가 생길 수 있음.
-보안 측면에서 약함
